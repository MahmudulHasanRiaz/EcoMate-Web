# Analytics Domain

> **Status:** Draft  

## Models

- `PageView` — Tracked page view in storefront

## Collection

- Auto-collected from storefront
- Displayed via dashboard widgets

## Owns

- Page view tracking
- Dashboard reporting widgets
- Marketing attribution data (future)

## Depends On

- **Orders** — Order data feeds analytics
- **Products** — Product view data feeds analytics
- **Sales & Marketing** — Campaign attribution feeds analytics

## Does NOT Own

- Business operations data (reads only)
- Order lifecycle
- Product catalog
- Any mutable business data
