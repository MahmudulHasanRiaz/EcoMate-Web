# Frontend Conventions

> **Status:** Draft  

## Admin

- Next.js 16 + Shadcn UI + TanStack Query
- Features under `src/features/` by domain
- Route files under `src/app/op/`

## Storefront

- Next.js 16 Server Components preferred
- Minimal client interactivity
- See `apps/storefront/AGENTS.md` for detailed rules
